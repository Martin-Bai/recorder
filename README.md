# recorder
recoder for IE9+ &amp; chrome &amp; firefox

# 非原创（只是整合代码）
引用：  
[RecorderJs](https://github.com/mattdiamond/Recorderjs)

# 注意事项
html5获取客户端麦克风和摄像头要求安全的网络环境，所以要尽量在 HTTPS 的环境下。  
所以访问的时候 Chrome47 以上版本和 QQ 浏览器因为没有 HTTPS 支持会直接报错。
